### 会员弹窗-全局规则-固定受众

<!-- block:paragraph -->
会员优惠弹窗**仅对已登录店铺账号的访客触发**，未登录用户一律不弹出，与推送时间、展示频次、展示页面等配置叠加生效——即使其他条件都满足，未登录时仍不触发。后台在页面标题下用一行文案提示该规则（以设计稿为准）。

<!-- block:h3 -->
### 会员弹窗-全局规则-C 端触发判断流程

<!-- block:mermaid -->
```mermaid
sequenceDiagram
    participant V as 访客
    participant C as C 端
    participant S as 服务端

    V->>C: 进入页面
    C->>S: 查询当前用户登录态
    alt 未登录
        S-->>C: 未登录
        C-->>V: 不触发任何会员弹窗
    else 已登录
        S-->>C: 已登录
        C->>S: 查询会员弹窗活动列表（进行中）
        alt 无进行中活动 / 活动已暂停
            S-->>C: 无可用活动
            C-->>V: 不弹出
        else 有进行中活动
            S-->>C: 返回活动配置
            C->>C: 检查推送时间延迟
            C->>S: 查询该用户×该活动的展示频次与停止条件
            alt 已触发停止条件 或 本周期已达频次上限
                S-->>C: 命中限制
                C-->>V: 不弹出
            else 未触发限制
                S-->>C: 可继续推送
                C->>C: 检查折扣配置方式
                alt 自定义模式
                    C->>S: 查询绑定的折扣码是否仍有效
                    alt 折扣码无效（已过期 / 已关闭）
                        S-->>C: 无效
                        C-->>V: 不弹出（活动状态保持进行中，后台可加内容异常提示）
                    else 折扣码有效
                        S-->>C: 有效
                        C-->>V: 展示优惠码弹窗（兑换前态）
                    end
                else 智能推荐模式
                    C->>S: 从可选折扣池中按推荐策略排序取码
                    alt 有可用推荐码
                        S-->>C: 返回推荐码
                        C-->>V: 展示优惠码弹窗（兑换前态）
                    else 无可用推荐码
                        C->>S: 查询兜底送积分活动是否有效
                        alt 兜底活动有效
                            S-->>C: 有效
                            C-->>V: 展示送积分兜底弹窗
                        else 兜底活动无效 / 未配置
                            S-->>C: 无效
                            C-->>V: 不弹出
                        end
                    end
                end
            end
        end
    end
```
